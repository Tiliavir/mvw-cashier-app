## Nutzung auf eigene Gefahr

Diese Anwendung ist ein privates Hobbyprojekt und wird ohne jegliche Gewährleistung
bereitgestellt. Die Nutzung erfolgt auf eigene Gefahr.

## Keine Haftung

Der Betreiber übernimmt keinerlei Haftung für Schäden, die durch die Nutzung oder
Nichtnutzbarkeit dieser Anwendung entstehen. Dies gilt insbesondere für:

- Verlust oder Beschädigung von Daten
- Fehler bei der Berechnung von Beträgen
- Ausfälle oder Fehlfunktionen der Anwendung

## Datenspeicherung

Alle Daten werden ausschließlich lokal im Browser (localStorage) gespeichert.
Es werden keine Daten an Server übertragen oder gespeichert.
